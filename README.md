# Dada Cafe

Static café site built with **HTML**, **CSS**, and **JavaScript** (no React or TypeScript).

## Structure

```
index.html      — page markup
css/styles.css  — layout and theme
js/data.js      — menu items and prices
js/cart.js      — cart state
js/main.js      — navbar, menu tabs, interactions
assets/         — images
```

## Run locally

ES modules need a local server (opening `index.html` directly may block imports).

```bash
npm run dev
```

Then open http://localhost:3000

Or use any static server, for example:

```bash
python3 -m http.server 3000
```
